# 💕 couple_diary_app
연인과의 평생의 사랑을 원하는 당신에게 필요한 앱!
서로에게 진솔한 이야기를 공유하세요.
- 여러가지 사랑에 관한 질문에 대해 답변을 등록합니다.
- 등록한 답변은 연인과 나만 공유할 수 있습니다.
